# Project Atlas Vision

Land Opportunity Finder exists to help developers discover land and property opportunities others miss by combining evidence, memory and AI into a trusted acquisition intelligence platform.

## Product principles

1. Find what others miss.
2. Evidence before opinion.
3. Never present uncertainty as fact.
4. Explain every recommendation.
5. Learn from investigations and outcomes.
6. Think nationally, build locally.
7. Prioritise verified opportunity quality over record volume.

## North star

Verified high-quality acquisition opportunities created.
